# Weather App

## Tech Stack
- React.js
- Tailwind CSS

## Features

### 1. Current Temperature
   - Get the current temperature by searching for a city.

### 2. Next 24 Hours Forecast
   - View the forecast for the next 24 hours, updated every 3 hours.
